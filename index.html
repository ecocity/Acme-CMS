Acme-CMS by ecocity

Multisite website management system and contact manager written in simple to use php and mysql.

Developing a simple and useful modern codebase that can be added to and configured as needed, easily.

Dependencies

php, mysql, jquery

Install

unzip package into web folder set up mysql database and import acmecms.sql file configure paths in db.inc file
